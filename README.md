# Groundwrk Starter Theme

A clean Gutenberg-first WordPress starter theme scaffold aligned to the Groundwrk developer guidelines.

## Included

- starter theme structure
- `theme.json` design system setup
- wide/full width support
- no page builders
- custom Gutenberg blocks
- inline editable block content
- accessible skip link and basic responsive navigation
- lightweight CSS/JS structure

## Install

1. Upload the `groundwrk-theme` folder to `/wp-content/themes/`
2. Activate the theme in WordPress
3. Go to Appearance → Menus and assign menus to `Primary Menu` and `Footer Menu`
4. Create/edit pages using the Block Editor
5. Use the custom blocks:
   - Groundwrk Hero
   - Groundwrk Section

## Notes

This is a production-ready starter scaffold, not a final client theme. It is intentionally lightweight so additional custom blocks, patterns, templates, CPT logic, and design details can be added cleanly.
